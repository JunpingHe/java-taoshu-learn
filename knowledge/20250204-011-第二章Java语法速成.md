### 2.1 变量与数据类型

**核心知识点**：
- Java区分**基本类型**和**包装类型**
- 基本类型：`int`, `double`, `boolean` 等（不能为null）
- 包装类型：`Integer`, `Double`, `Boolean` 等（可以为null）
- `String`永远是包装类型

**为什么这么设计**？
```
基本类型 = 实体商品（必须有货）
包装类型 = 商品券（可以是空的）
```

| Python | Java基本类型 | Java包装类型 |
|--------|-------------|-------------|
| int    | int         | Integer      |
| float  | double      | Double       |
| bool   | boolean     | Boolean      |
| str    | 无          | String       |

### 2.2 字符串操作

**核心知识点**：
- Java的`String`是大写的（类名）
- 字符串不可变
- `==`比较引用，`.equals()`比较内容
- 大量拼接用`StringBuilder`，别用`+`

**性能陷阱**：
```java
// ❌ 错误：每次创建新对象
String s = "";
for (int i = 0; i < 10000; i++) {
    s += i;
}

// ✅ 正确：使用StringBuilder
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 10000; i++) {
    sb.append(i);
}
String s = sb.toString();
```

### 2.3 条件与循环

**核心知识点**：
- if语句必须用大括号 `{}`，即使单行
- for循环有三种写法：传统for、增强for、Stream

**最佳实践**：
```java
// ✅ 永远用大括号
if (age >= 18) {
    System.out.println("成年");
}

// 三种for循环
// 方式1：传统for（需要索引）
for (int i = 0; i < list.size(); i++) { }

// 方式2：增强for
for (String item : list) { }

// 方式3：Stream
list.forEach(item -> System.out.println(item));
```

### 2.4 函数与方法

**核心知识点**：
- Java方法必须在类中（纯面向对象）
- 静态方法用`static`，直接通过类调用
- 实例方法需要创建对象
- 支持方法重载（同名不同参）

```java
// 静态方法
public static int add(int a, int b) {
    return a + b;
}
// 调用：Utils.add(1, 2)

// 方法重载
public String greet(String name) { }
public String greet(String name, Integer age) { }
```

### 2.5 面向对象

**核心知识点**：
- 接口`interface`定义"能做什么"
- 类用`implements`实现接口
- 一个类可以实现多个接口（解决单继承限制）

```java
public interface Flyable {
    void fly();
}

public class Bird implements Flyable {
    @Override
    public void fly() {
        System.out.println("鸟在飞");
    }
}
```

### 2.6 集合框架

**核心知识点**：
- `List`：允许重复，有序（ArrayList）
- `Set`：不允许重复（HashSet）
- `Map`：键值对（HashMap）

```java
// List
List<String> list = new ArrayList<>();
list.add("书1");

// Set
Set<String> set = new HashSet<>();
set.add("书1");  // 自动去重

// Map
Map<String, Integer> map = new HashMap<>();
map.put("书1", 88);
```

### 2.7 异常处理

**核心知识点**：
- 受检异常必须声明`throws`或处理`try-catch`
- 强制开发者关注错误

```java
// 必须声明可能抛出的异常
public String readFile(String filename) throws IOException {
    // ...
}

// 调用者必须处理
try {
    String content = readFile("file.txt");
} catch (IOException e) {
    e.printStackTrace();
}
```

---

### 2.8 Lambda与Stream

**核心知识点**：
- Java 8+支持函数式编程
- Stream支持链式操作

```java
// 筛选偶数并求平方
List<Integer> result = numbers.stream()
    .filter(n -> n % 2 == 0)
    .map(n -> n * n)
    .collect(Collectors.toList());
```

## 🎯 核心差异总结

| 特性 | Python | Java |
|------|--------|------|
| 类型系统 | 动态、宽松 | 静态、严格 |
| null处理 | None是正常值 | null要特殊对待 |
| 函数定义 | 可以独立 | 必须在类中 |
| 重载 | 不支持 | 支持 |
| 接口 | 抽象类 | 必须有接口 |
| 异常 | 不声明 | 受检异常必须声明 |
