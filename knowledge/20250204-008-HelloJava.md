# 第一个Java程序

## 📅 日期：2025-02-04

## 🎯 学习目标
编写并运行第一个Java程序 - Hello Java

## 📝 代码结构

```java
package org.example;  // 包声明：告诉Java这个类在哪个包里

public class HelloJava { // 类声明：所有Java代码都在类中
    public static void main(String[] args) { // 主方法：程序入口
        System.out.println("Hello, Java!"); // 输出：在控制台打印文字
    }
}
```

## 🔑 关键知识点

### 1. 类（Class）
- Java是面向对象的语言，所有代码都必须写在类中
- 类名首字母大写（驼峰命名）

### 2. main方法
```java
public static void main(String[] args)
```
- `public`：公开的，任何地方都能访问
- `static`：静态的，不需要创建对象就能调用
- `void`：没有返回值
- `main`：方法名，JVM会找这个方法作为程序起点
- `String[] args`：命令行参数（数组）

### 3. 输出语句
```java
System.out.println("内容");  // 打印后换行
System.out.print("内容");    // 打印不换行
```

## 🚀 运行方式

### Maven命令
```bash
mvn compile                    # 编译
mvn exec:java -Dexec.mainClass="org.example.HelloJava"  # 运行
```

### IDEA快捷方式
点击代码旁的绿色 ▶️ 按钮

## 💡 常见问题

| 问题 | 解决方案 |
|------|---------|
| 找不到main方法 | 检查拼写：`main` 不是 `Main` |
| 中文乱码 | 确保文件编码是UTF-8 |
| 包名错误 | 确保文件在 `src/main/java/org/example/` 目录下 |
