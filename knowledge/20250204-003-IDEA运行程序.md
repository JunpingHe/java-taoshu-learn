# 20250204-IDEA运行程序

## 📋 知识点总结

### 1. IDEA 运行 Java 程序的四种方式

| 方式 | 操作 | 适用场景 |
|------|------|----------|
| **绿色按钮** | 点击代码旁的 ▶ | 快速运行当前文件 |
| **右键菜单** | 右键 → Run | 鼠标操作习惯者 |
| **快捷键** | `Shift + F10` | 键盘流，提高效率 |
| **运行配置** | 右上角 ▶ | 需要设置参数时 |

### 2. 运行配置（Run Configuration）

什么时候需要创建运行配置？
- ✅ 需要传递命令行参数
- ✅ 需要设置 JVM 参数（如内存大小）
- ✅ 需要设置环境变量
- ✅ 频繁运行同一个程序

配置内容：
```xml
Main class: org.example.Main          # 主类全限定名
VM options: -Xmx512m                  # JVM参数（可选）
Program arguments: arg1 arg2           # 命令行参数（可选）
```

### 3. main 方法详解

```java
public static void main(String[] args) {
    // 代码
}
```

| 关键字 | 作用 |
|--------|------|
| `public` | 公开的，JVM可以调用 |
| `static` | 静态的，不需要创建对象就能调用 |
| `void` | 无返回值 |
| `main` | 方法名，JVM固定查找这个名字 |
| `String[] args` | 命令行参数数组 |

### 4. 程序退出码

```
退出代码为 0  → 正常结束
退出代码非0   → 异常结束
```
