## 📈 学习建议

### 优先级1：补充核心语法实践
建议编写以下练习代码：

1. **基本类型 vs 包装类型**
   ```java
   int a = 10;        // 基本类型
   Integer b = 20;    // 包装类型
   Integer c = null;  // 包装类型可以为null
   ```

2. **StringBuilder vs 字符串拼接**
   ```java
   StringBuilder sb = new StringBuilder();
   sb.append("Hello").append(" ");
   sb.append("Java");
   ```

3. **for循环**
   ```java
   // 传统for
   for (int i = 0; i < 10; i++) { }

   // 增强for
   List<String> list = new ArrayList<>();
   for (String item : list) { }
   ```

4. **接口**
   ```java
   public interface Flyable {
       void fly();
   }

   public class Bird implements Flyable {
       @Override
       public void fly() {
           System.out.println("鸟在飞");
       }
   }
   ```

### 优先级2：补充集合框架
```java
// List
List<String> books = new ArrayList<>();
books.add("Java入门");
books.add("Python精通");

// Set
Set<String> uniqueBooks = new HashSet<>();

// Map
Map<String, Integer> prices = new HashMap<>();
prices.put("Java入门", 88);
```

### 优先级3：补充Lambda与Stream
```java
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// 筛选偶数
numbers.stream()
    .filter(n -> n % 2 == 0)
    .forEach(System.out::println);
```
